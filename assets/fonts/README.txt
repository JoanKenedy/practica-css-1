Fuente variable DM Sans
=====================

Esta descarga contiene DM Sans tanto en forma de fuentes variables como de fuentes estáticas.

DM Sans es una fuente variable con estos ejes:
opsz
wght

Esto significa que todos los estilos están contenidos en estos archivos:
DMSans-VariableFont_opsz,wght.ttf
DMSans-Italic-VariableFont_opsz,wght.ttf

Si su aplicación es totalmente compatible con fuentes variables, ahora puede elegir estilos intermedios
que no están disponibles como fuentes estáticas. No todas las aplicaciones admiten fuentes variables y
en esos casos, puedes usar los archivos de fuentes estáticas para DM Sans:
static/DMSans-Regular.ttf
static/DMSans-Medium.ttf
static/DMSans-MediumItalic.ttf

Comienza
-----------

1. Instala los archivos de fuentes que quieras usar

2. Usa el selector de fuentes de tu aplicación para ver la familia de fuentes y todos los
estilos disponibles
